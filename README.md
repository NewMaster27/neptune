# [ archnon-project ]
[!] Automation/Script of tor as a secure proxy on systemd based linux distros.

[!] Use: git clone https://github.com/NewMaster27/Archnon-project && cd Archnon-project && mv archnon.git archnon && mv archnon /usr/bin && chmod +x archnon && archnon help

[!] In addition: 
- Recommended browsers: falkon, surf, torbrowser, librewolf, badwolf.
- Recommended: MAC address constant changing.
- Recommended: Use an aditional proxy: proton, mullvad, etc.

[+] Enjoy it !!

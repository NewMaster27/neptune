# [ archnon-project ]
[!] Automation/Script of tor as a secure proxy on systemd based linux distros.

[!] Use: git clone https://github.com/NewMaster27/Archnon-project
cd Archnon-project 
chmod +x make_bin.sh 
sudo ./make_bin.sh

[!] Use: wget https://github.com/NewMaster27/Archnon-project/archnon.git && wget https://github.com/NewMaster27/Archnon-project/make_bin.sh && chmod +x make_bin.sh && sudo ./make_bin.sh

[!] In addition: 
- Recommended browsers: falkon, surf, torbrowser, librewolf, badwolf.
- Recommended: MAC address constant changing.
- Recommended: Use an aditional proxy: proton, mullvad, etc.

[+] Enjoy it !!
